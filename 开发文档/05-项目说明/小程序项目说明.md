# 小程序项目说明

当前目录是企业管理助手的小程序项目：

```text
boss-chat-miniapp
```

技术栈：

```text
Vue 3 + uni-app + TypeScript
```

目标平台：

```text
微信小程序、支付宝小程序
```

## 当前已经具备

- 登录页
- 登录成功后的占位首页
- 与后端 `/api/auth/login`、`/api/auth/me`、`/api/auth/logout` 的联调代码
- 微信小程序和支付宝小程序构建脚本
- H5 预览端口 `9900`

## 本地 H5 预览

```powershell
cd D:\LanTu\life-cycle-management-system\bossChat\boss-chat-miniapp
npm install
npm run dev:h5
```

默认访问：

```text
http://localhost:9900
```

> `9900` 只是 H5 调试预览端口。真正运行在微信或支付宝里的小程序，并不是通过这个本地端口对外提供服务。

## 小程序构建

微信小程序：

```powershell
npm run dev:mp-weixin
```

默认开发命令：

```powershell
npm run dev
```

它等价于：

```powershell
npm run dev:mp-weixin
```

支付宝小程序：

```powershell
npm run dev:mp-alipay
```

构建产物会输出到 `dist/dev` 下对应的平台目录。  
对于微信小程序，执行 `npm run dev:mp-weixin` 后，打开：

```text
dist/dev/mp-weixin
```

即可在微信开发者工具中正常启动、预览和调试。

## 接口地址

开发环境默认接口地址：

```text
http://localhost:9090/api
```

配置位置：

```text
.env.development
```

在真机或正式小程序环境里，`localhost` 不再指向你的电脑。到那一步时，需要把接口地址改成可访问的局域网地址或正式 HTTPS 域名，并按各平台要求配置合法请求域名。
